Sintomas

Quando você usa o Auxiliar de Trabalho pela primeira vez, é necessário se cadastrar no sistema. Após o cadastro, a tela de inicialização congela e não permite continuar.

Causa

Este erro ocorre por causa de um bug no Auxiliar de Trabalho. Existem campos no formulário que são obrigatórios, mas o módulo de cadastro não informa todas as varíaveis necessárias e o cadastro não é validado pelo servidor de dados.

Resolução

Verifique se já existe uma nova versão do Auxiliar de Trabalho. Para isto, basta executar o Atualizador do Auxiliar. Caso ainda não exista uma versão nova, tente o método abaixo:

O usuário deve enviar um email para douglas.dealmeida@previdencia.gov.br com os dados cadastrais para que o cadastro seja feito diretamente no servidor.

As versões futuras do Auxiliar de Trabalho corrigirão este erro.

Resolução

Este é um problema no Auxiliar de Trabalho beta8 versão 1.0.0424.

Este é um problema no Auxiliar de Trabalho beta8a1 versão 1.0.0430.

Este problema foi corrigido no Auxiliar gama1 versão 1.0.0551.
