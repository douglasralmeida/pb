Introdução

Este artigo informa como usar o Auxiliar versão Treinamento e as senhas de acesso de cada usuário-modelo.

Informações

O Auxiliar versão Treinamento é uma versão especial do Auxiliar que pode ser usada para realização de treinamentos, testes e homologações. Os dados do servidor, para esta versão, são fícticios e não devem ser usados em nenhum processo do âmbito do INSS.

Senhas de acesso

Para acessar o Auxiliar versão Treinamento use as credenciais dos usuários-modelo. Um usuário-modelo possui previlégios idênticos a todos os usuários pertencentes ao grupo/setor da APS que ele participa/trabalha.

Não tente usar a senha de acesso do Auxiliar comum para acessar o Auxilair versão Treinamento, ou vice-versa.
