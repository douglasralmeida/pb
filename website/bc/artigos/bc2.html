Sintomas

Quando você agenda um segurado, existe a opção para imprimir o comprovante do agendamento. Entretanto, o comprovante é impresso com a seguinte frase: "Informamos que foi agendado algo desconhecido no local...". Além disso, o nome do cliente não é impresso.

Causa

Este erro ocorre por causa de um bug no motor de relatórios do Auxiliar de Trabalho. Ele será corrigido na próxima versão do aplicativo.

Resolução

Verifique se já existe uma nova versão do Auxiliar de Trabalho. Para isto, basta executar o Atualizador do Auxiliar. Caso ainda não exista uma versão nova, tente o método abaixo.

Imprima uma 2ª via do comprovante de marcação, que sairá com o texto correto. Faça uma pesquisa com o nome do segurado. No resultado da pesquisa, verifique qual correponde ao segurado em questão e clique no link Imprimir Comprovante de Marcação.

Situação

Este é um problema no Auxiliar de Trabalho beta7 versão 1.0.0373

Este problema foi corrigido no Auxiliar gama1 versão 1.0.0551.
