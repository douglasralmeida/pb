Sintomas

Ao tentar abrir um formulário do Modelos de Documentos diretamente pelo navegador Internet Explorer você pode encontrar algum dos problemas descritos abaixo:

  • Metade da área do navegador fica preenchido por um quadro branco inutilizável.
  • Você não consegue alterar o texto do formulário.
  • É necessário clicar com o botão direito do mouse numa área do texto e, em seguida, clicar na opção Editar.
  • O formulário não é exibido pelo navegador.
  • O Internet Explorer para de responder ou fecha automaticamente.

Causas

Os formulários OpenDocumento necessitam do Openoffice para serem editados, entranto, o Internet Explorer possui vários problemas de compatibilidade com o OpenOffice. É recomendável abrir os formulários usando uma janela separada do OpenOffice.

Soluções

Para resolver este problema é necessário desativar o controle ActiveX do OpenOffice que é usado pelo Internet Explorer para exibir os documentos no formato OpenDocument. Depois de removido, ao clicar num atalho para um formulário do Modelos de Documentos, o Internet Explorer perguntará qual ação será feita com o documento e, assim, abrirá o formulário em uma janela separada do OpenOffice.

Corrigir sozinho:

Apague do Registro do Windows as seguintes chaves:

  • HKEY_CLASSES_ROOT\CLSID\{61FA3F13-8061-4796-B055-3697ED28CB38}
  • HKEY_CLASSES_ROOT\CLSID\{67F2A879-82D5-4A6D-8CC5-FFB3C114B69D}

Mais informações

Para reverter as alterações feitas, reinstale o OpenOffice novamente.
