Introdução

A Microsoft liberou no dia 14/03/2011 a nona versão do navegador Internet Explorer. O aplicativo já está disponível para download no Portal da BHSul e, em breve, estará disponível para atualização no Windows Update para todos os computadores com Windows Vista ou Windows 7.

Problemas

O Internet Explorer 9 possui várias alterações no seu motor de renderização. Essas alterações podem causar problemas de compatibilidade ao acessar as páginas da intranet da Previdência Social.

As seguints páginas apresentaram incompatibilidades

  • Modo de Administração do Portal da BH Sul
  • SIPPS
  • SICAU

Também foram encontrados problemas com o script do servidor proxy da intranet do INSS.

Soluções

Para resolver o problema de renderização, ao acessar a página da web pela primeira vez, você deve adicioná-la a lista de compatibilidade do Internet Explorer. Assim, o navegador usará o motor antigo para renderizá-la.

Quando o administrador da página lançar uma nova atualização compatível com o Internet Explorer 9, você deverá removê-la da lista de compatibilidade.

Para resolver o problema com o script do servidor proxy, instale o script da Dataprev novamente. Após a instalação, desmarque a opção "Detectar automaticamente as configurações" da caixa de diálogo Configurações da Rede Local (LAN). Esta caixa de diálogo está na guia Conexões da caixa Opções da Internet.
