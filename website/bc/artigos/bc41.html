Introdução

Este artigo descreve como determinar manualmente se um computador está usando a versão de 32 bits ou de 64 bits do sistema operacional Windows. Geralmente, a contagem de bits de um computador indica quantos dados ele pode processar, a velocidade com a qual ele processa os dados e a capacidade máxima da memória. Durante a instalação de um aplicativo, é recomendável usar a versão apropriada para o sistema operacional correspondente em bits

Determinar manualmente qual versão do Windows está instalada

O método abaixo é compatível com Windows Vista, Windows 7, Windows 8.1 Update e Windows 10.

  1. Clique em Iniciar.
  2. Digite sistema na caixa de Pesquisa.
  3. Clique em Sistema na lista de Programas.
  4. Procure, na tela, o item Tipo de Sistema, situado no grupo Sistema.
    • Se aparecer Sistema operacional de 32 bits, significa que o sistema operacional instalado é de 32 bits.
    • Se aparecer Sistema operacional de 64 bits, significa que o sistema operacional instalado é de 64 bits.
