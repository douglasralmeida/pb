Sintomas

Após se conectar no Auxiliar de Trabalho, você recebe várias mensagens de erro no soquete TCP com a seguinte descrição: "Um erro interno no soquete TCP impediu o Auxiliar de Trabalho de executar uma operação interna. Tente reiniciar o Auxiliar de Trabalho para resolver o problema. ID:049".

Causa

Este erro ocorre por causa de um erro do Auxiliar.

Resolução

  1. Abra o Painel de Controle e vá para Adicionar ou Remover Programas.
  2. Remova o Auxiliar de Trabalho.
  3. Acesse a página da BhSul digitando www-bhsul no navegador da Internet.
  4. Baixe a vesão mais atual do Auxiliar, versão gama1.
  5. Instale o Auxiliar gama1.

Este é um problema no Auxiliar de Trabalho beta8 versão 1.0.0424.

Este é um problema no Auxiliar de Trabalho beta8a1 versão 1.0.0430.

Este problema foi corrigido no Auxiliar gama1 versão 1.0.0551.
